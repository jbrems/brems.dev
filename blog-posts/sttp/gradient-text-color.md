---
title: 'How to set a gradient text color'
description: 'Straight to the point instructions on how to set a gradient text color.'
date: 2024-10-27
---

```css
.your-text-element {
  background: linear-gradient(-10deg, transparent, goldenrod, gold, goldenrod, transparent);
  background-clip: text;
  color: transparent;
}
```
